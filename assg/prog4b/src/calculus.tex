The partial integration theorem states that:

\[
  \int u(x) v'(x)\ dx = u(x) v(x) - \int v(x) u'(x)\ dx.
\]

Suppose that $f(x) = x\ cos(x)$.  Then we may choose:

\begin{center}
\vbox{
\begin{itemize}
  \item  $u=x$
  \item  $dv=cos(x)$ 
  \item  $du=1$ 
  \item  $v=sin(x)$
\end{itemize}
}
\end{center}

Following the formula, 

\[
  \int x\ cos(x)\ dx = x\ sin(x) - \int sin(x)\ dx,
\]

which reduces to

\[
  \int x\ cos(x)\ dx = x\ sin(x) - cos(x) + C. 
\]
